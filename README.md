# --- PYTHON SOULS ---

Se trata de un mini-juego por turnos, desarrollado con Python y manejado desde CLI. 

## La aplicación ofrece:

    1. ¿Un rato de entretenimiento?
    2. Jugabilidad simple pero exigente (y un poco de RNG)
    3. Una mini-historia en tres capítulos:
        4. Introducción
        5. Palacio
        6. Desafío
